
# Pecin Design — Portfólio (Astro + Tailwind)

## Rodando localmente
```bash
npm install
npm run dev
```

## Build e preview
```bash
npm run build
npm run preview
```

## Deploy no Cloudflare Pages
- Build command: `npm run build`
- Output directory: `dist`

Caso veja erros de dependência, verifique se as versões estão como no `package.json`.
